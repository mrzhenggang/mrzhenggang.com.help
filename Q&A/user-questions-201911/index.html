    1. 用户问题解答 2019-11¶

    1.1 计算节点readlink命令找不到¶

    1.1.1 问题描述¶

    在计算节点运行脚本，slurm.out报错readlink命令找不到

    1.2 解决方法¶

    登陆到登陆节点，如ln2，然后加载对应的模块，如 module add loginnode/ln2，使用命令检查 readlink 是否存在

    $ module add loginnode/ln2 # 注意是登陆ln2节点，所以加载ln2的
$ yhrun -N 1 -n 1 -p debug which readlink
/vol-th/software/loginnode/ln2/usr/bin/readlink
